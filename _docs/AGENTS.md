# `_docs/` 디스크립터

> 코드로 표현할 수 없는 지식의 보관 영역. 워크로그와 4개 인식론적 카테고리로 구성된다.
> 새 문서의 카테고리 판별 기준은 [AGENTS/how-to-separate-docs-folders.md](../AGENTS/how-to-separate-docs-folders.md) 참조.

## 하위 폴더

| 폴더 | 담는 것 |
|------|---------|
| [_worklog/](_worklog/AGENTS.md) | 작업 상태 운영 메타 (작업 트리, 상태 레지스트리) |
| [_ontology/](_ontology/AGENTS.md) | 도메인 개념 정의 ("무엇이 존재하는가") |
| [_knowledge/](_knowledge/AGENTS.md) | 외부 사실, 환경, 벤치마크 ("세상이 어떠한가") |
| [_strategy/](_strategy/AGENTS.md) | 우리의 결정과 근거 ("왜 이렇게 하기로 했는가") |
| [_architecture/](_architecture/AGENTS.md) | 설계 명세 ("어떻게 만드는가") |
