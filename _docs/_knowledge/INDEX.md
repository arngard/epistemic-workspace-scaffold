# 지식 색인

> "세상이 어떤가" — 외부 사실, 환경 정보, 벤치마크 분석.
> 우리의 결정과 그 근거는 [_strategy/](../_strategy/INDEX.md) 참조.
> 최종 설계 명세는 [_architecture/](../_architecture/INDEX.md) 참조.
> 공통 사용 규칙은 [AGENTS.md](../../AGENTS.md)의 "색인(INDEX) 사용 규칙" 섹션 참조.

## 작성 스타일

- 출처와 발견 일자를 반드시 명시한다.
- 서술 주체는 "우리"가 아니라 "세상". "X는 Y다" 형식의 단정문 또는 관찰형 진술로 쓴다.
- 우리의 결정이나 근거는 포함하지 않는다. 그건 `_strategy/`에 둔다.
- 단위 문서는 한 페이지 이내가 권장이다.
