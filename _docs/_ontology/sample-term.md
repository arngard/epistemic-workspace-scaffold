# (샘플) 용어 정의

> ⚠️ 이 파일은 스캐폴드가 제공하는 튜토리얼 샘플이다. INDEX에 등록된 예시로 두었다.
> 실제 내용 작성 시 이 파일을 삭제하고, 의미 있는 이름의 새 파일로 교체한 뒤 [INDEX.md](INDEX.md)를 갱신한다.

이 카테고리에는 "같은 단어를 두 사람(또는 두 문서)이 다른 뜻으로 쓸 수 있는 도메인 용어"를 정의한다. 보편적 용어는 정의하지 않는다. 예: "사용자"가 호스트 앱의 사용자인지 SDK 연동자인지를 명시해야 할 때.
